# ReactiveTest

The main idea of the project is to list trending projects of Github and when tap on one of them, show their details.

#Required

    Swift 2.3, Xcode 7.3
    iOS 9+



#Usage

For Installing it on Swift 2.3 with CocoaPods, specify it in your 'Podfile'.
```

platform :ios, '9.0'
use_frameworks!
pod 'ReactiveCocoa'
pod 'MBProgressHUD'

Run 'pod install'
